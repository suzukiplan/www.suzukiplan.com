VGS-Zero

VGVGS-Zero is a game console with 16MHz Z80, proprietary VDP, and proprietary chip-tuned sound source VGS (Video Game Sound), running on a RaspberryPi Zero 2W bare metal environment.
It can also be run as an application on a PC (Windows, macOS, Linux).
See the details on the GitHub.
